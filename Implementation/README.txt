# Pyon distribuído sob py_netgames

## Para executar este programa

## Ter seu ambiente de programação previamente configurado (considerando VS Code)

1. ter instalada a extensão "Python" publicada pela Microsoft
2. ter instalada a biblioteca pipenv (pip install pipenv, para instalar, apenas 1 vez)

## Preparação para usar Pyng na execução (assim como em um desenvolvimento)

1. python -m pipenv install (criar um ambiente)
2. no VS Code, ctrl+shift+p, select interpreter, seleciona o marcado com 'PipEvn'
3. python -m pipenv shell (para iniciar um terminal)

## Execução do jogo da velha

- python -m pyon (para rodar o programa)
